계정 삭제 안내

1. 계정 삭제 방법

이용자는 언제든지 계정 삭제를 요청할 수 있습니다. 계정 삭제를 원하실 경우, 다음 방법 중 하나를 이용해 주세요.

  • 앱 내 경로: [프로필] → [설정] → [계정 삭제]
  • 이메일 요청: 등록된 이메일 주소에서 yellagoya@kakao.com 으로 계정 삭제 요청 메일 발송

2. 삭제되는 데이터의 범위

계정 삭제 시 다음과 같은 데이터가 삭제됩니다.

  • 회원 가입 시 수집된 이메일 주소, 소셜 로그인 ID(구글, 카카오)
  • 사용자가 작성한 모든 컨텐츠(투두, 다이어리, 댓글 등)
  • 기타 계정과 직접적으로 연관된 모든 개인정보

3. 데이터 보관 기간

계정 삭제 요청이 접수되면, 관련 법령에 따라 최대 30일 이내에 모든 데이터가 완전히 삭제됩니다. 단, 법적 의무 준수를 위해 일정 기간 보관이 필요한 경우 해당 기간 동안 안전하게 보관 후 파기합니다.

4. 일부 데이터 삭제 요청

계정을 삭제하지 않고도 앱 내에서 개별 컨텐츠(투두, 다이어리 등)를 직접 삭제할 수 있습니다. 삭제된 데이터는 즉시 서비스에서 확인할 수 없으며, 최대 30일 이내에 서버에서 완전히 삭제됩니다.
